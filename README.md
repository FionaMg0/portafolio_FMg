# Fiona Mondelo | Portafolio Personal 

Bienvenido a mi portafolio personal. Este es un espacio diseñado para mostrar mis proyectos, habilidades y pasión por el desarrollo de aplicaciones multiplataforma y el diseño UI/UX.

## Características

- **Diseño Minimalista y Premium**: Una interfaz limpia, moderna y elegante.
- **Terminal Interactiva**: Una experiencia retro-tecnológica para explorar mis habilidades mediante comandos.
- **Carrusel de Proyectos**: Visualización dinámica de mis trabajos más recientes.
- **Modo Oscuro/Claro**: Adaptabilidad total según la preferencia del usuario.
- **Despliegue Automático**: Configurado con GitHub Actions para actualizaciones instantáneas.

## Tecnologías Utilizadas

- **Core**: [Vue.js 3](https://vuejs.org/) (Composition API)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Despliegue**: [GitHub Pages](https://pages.github.com/) + Actions
- **Tipografía**: Inter & Playfair Display
